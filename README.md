# Credit Card Fraud Detection - Exploratory Data Analysis

## 📌 Project Overview
Performed EDA and Data Visualization on a real-world credit card transaction dataset to identify fraud patterns and transaction behaviors.

## 🛠️ Tools & Libraries
- Python, Pandas, Matplotlib, Seaborn
- Jupyter Notebook

## 📊 Dataset
- Source: Kaggle - Credit Card Fraud Detection
- 284,807 transactions | 31 features
- Sampled: 10,000 rows for analysis

## 🔍 Key Findings
- Dataset is highly imbalanced — only 0.16% transactions are fraudulent
- Most transactions (normal & fraud) involve small amounts under $200
- Fraud transactions show high-value outliers — fraudsters target large amounts
- Fraud activity concentrated in the first half of the time period

## 📈 Visualizations
- Fraud vs Non-Fraud transaction count
- Transaction amount distribution
- Transaction time by class
- Amount by class (boxplot)

- `fraud_analysis.png` — Visualization output
